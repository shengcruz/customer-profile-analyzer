# 客户画像分析器 - 产品需求文档 (PRD)

## 1. 产品概述

一个专为五恒空调系统销售人员设计的客户分析工具。通过分析潜在客户的朋友圈截图，利用 AI 生成客户画像和精准销售策略。

### 1.1 产品定位

| 项目 | 说明 |
|-----|------|
| 目标用户 | 五恒空调系统销售人员 |
| 分析对象 | 潜在客户的朋友圈 |
| 输出目标 | 客户画像 + 销售策略 + 话术建议 |

### 1.2 五恒系统介绍

五恒空调系统是高端舒适家居解决方案，包含：
- **恒温**：全年保持舒适温度（20-26°C）
- **恒湿**：智能调节湿度（40-60%）
- **恒氧**：24小时新风换气，保持空气清新
- **恒洁**：多重过滤，去除PM2.5、甲醛等污染物
- **恒静**：低噪音运行，不影响睡眠和生活

## 2. 核心功能

### 2.1 图片上传
- 支持多图上传（建议5张以上）
- 支持拖拽上传
- 支持常见格式：JPG、PNG、WEBP
- 上传后显示缩略图预览

### 2.2 AI 分析
- 调用 DeepSeek API 进行多模态分析
- 支持用户输入自己的 API Key
- 分析朋友圈内容，生成客户画像

### 2.3 报告生成
- Markdown 格式报告展示
- 支持复制报告文本
- 支持下载报告

### 4.2 交互流程
1. 用户输入 DeepSeek API Key
2. 用户上传多张朋友圈截图
3. 点击"生成销售策略"
4. 显示分析进度
5. 展示分析报告
6. 支持复制或下载报告

## 5. 技术要求

- 纯前端实现（HTML + CSS + JavaScript）
- 调用 DeepSeek API（OpenAI 兼容格式）
- 图片转 Base64 发送
- marked.js 渲染 Markdown
- 响应式设计，支持移动端

## 6. 样式规范
